== README

Ruby version:  2.5.5

Prerequisite: Set up environment variables

Running the project:

* vagrant up
* vagrant ssh
  ** cd /vagrant
  ** bundle install
  ** bundle exec rake db:setup
  ** bundle exec rake db:seed
  ** bundle exec rails console -> create one or more users
  ** bin/rails server -b 0.0.0.0
* vagrant halt || vagrant suspend
* vagrant destroy
